# iSchool ERP

A Django-based Institute Management ERP system.

## Features

- Student Enrollment
- Batch Management
- Attendance Tracking
- Fee Management
- Student Portal

## Tech Stack

Backend: Django + DRF  
Database: PostgreSQL  
Frontend: Bootstrap  

## Project Structure

apps/
accounts/
students/
courses/
attendance/
fees/

## Setup

git clone repo
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
